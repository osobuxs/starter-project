# starterV2

Esta carpeta documenta la **versión real implementada** del proyecto después de la evolución del starter original.

## Fuente de verdad actual

Para entender cómo funciona hoy el sistema, tomá como canónicos los documentos dentro de `starterV2/`.

## Contenido

- `brief/` — alcance implementado y ajustes respecto del starter original
- `backend/` — contrato real de Firestore/Storage, reglas e índices actuales
- `frontend/` — política de navegación, auth gating y organización vigente
- `LEGACY_NOTES.md` — contexto sobre la diferencia entre el starter original y la versión actual

## Qué no reemplaza

Los archivos legacy del root y `backend/` se mantienen como referencia histórica del challenge original, pero no describen necesariamente el estado actual del producto.
