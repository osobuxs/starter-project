# Legacy Notes

El starter original estaba orientado a noticias remotas y a una consigna base más acotada.

La implementación actual:
- reemplaza el feed remoto por Firestore
- agrega Google Sign-In
- agrega authoring real de notas
- agrega drafts, favoritos persistentes y profile photo storage

Por eso, documentación legacy en la raíz o en `backend/` puede no reflejar el estado real actual.

Usar `starterV2/` como fuente de verdad para la versión entregable actual.
